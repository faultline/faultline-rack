# Faultline::Rack

> [faultline](https://github.com/faultline/faultline) exception and error notifier for Rack application.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'faultline-rack'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install faultline-rack
